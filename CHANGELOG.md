# Changelog

All notable changes to **The Construct** are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/) and the project adheres to
[Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-06-04

### Added
- Initial release of the calm-cyberpunk theme family, generated from a
  zero-dependency design-token compiler.
- Four canvas depths: **Midnight**, **Void**, **Twilight**, **Daybreak**.
- Four chrome accents: **Aqua**, **Orchid**, **Synth**, **Acid**.
- Eight curated variants spanning the depth × accent matrix.
- Full workbench coverage, TextMate `tokenColors`, and LSP `semanticTokenColors`
  kept in lock-step.
- Build-time ergonomic contrast audit enforcing a readability floor and an
  anti-glare comfort ceiling on every variant.
